Problem und Nutzen

Aus umfangreichen Quellen wird eine strukturierte Arbeitsgrundlage.

Wenn Dokumente, Berichte oder E-Mail-Verläufe zu lang werden, fehlt oft der schnelle Blick auf Relevanz, Risiken und nächste Schritte.

SMART SummaryAssistent verdichtet diese Inhalte für typische Arbeitssituationen: verständlich, steuerbar und direkt weiterverwendbar, ohne jedes Mal eigene Prompts formulieren zu müssen.

Schnelle Lagebilder, Executive Summaries und klare Grundlagen für Entscheidungen.

Projektberichte

FAQ

Kurz beantwortet.

Zur App

Welchen Nutzen bringt mir SMART SummaryAssistent?

SMART SummaryAssistent macht aus langen Dokumenten, Berichten, E-Mails und Fachtexten klare Zusammenfassungen, belastbare Entscheidungsgrundlagen und sofort nutzbare Arbeitsergebnisse. Die App führt dich durch den Ablauf, sodass du nicht jedes Mal eigene Prompts formulieren musst.

Welche Vorteile habe ich bei der Nutzung?

Du arbeitest geführt statt frei zu prompten: Vorlage, Zielgruppe, Fokus, Ton, Länge und Ausgabeformat werden bewusst eingestellt. Dadurch entstehen konsistentere Ergebnisse, weniger manuelle Nacharbeit und wiederverwendbare Profile für ähnliche Aufgaben.

Für wen ist SMART SummaryAssistent gedacht?

Die App richtet sich an Selbstständige, Beratung, Management, Projektarbeit, Verwaltung, Fachabteilungen, Office und Wissensarbeit, wenn regelmäßig längere Dokumente, Berichte, E-Mails oder Fachtexte schnell erfasst und weiterverarbeitet werden müssen.

Welche Aufgaben unterstützt die App?

SMART SummaryAssistent unterstützt Zusammenfassungen, Executive Summaries, Entscheidungsnotizen, Stichpunkte, tabellarische Ausgaben, Risiko- und Projektübersichten sowie das spätere Kopieren, Exportieren oder Drucken der Ergebnisse.

Welche Rolle spielt KI in SMART SummaryAssistent?

KI ist Bestandteil der App-Funktion. Die Nutzung erfolgt über BuiltSmart AI Credits oder optional über einen eigenen Anthropic API Key. Im Trial sind 1.000 app-spezifische BuiltSmart AI Test-Credits enthalten.

Welche Inhalte kann ich in der App bearbeiten, speichern oder organisieren?

Du kannst Quellen, Zusammenfassungseinstellungen, Profile, Verlauf und Arbeitsergebnisse in der jeweiligen App bearbeiten. Soweit Inhalte gespeichert werden, bleiben sie lokal im Browser; Export und Import per JSON helfen bei Sicherung oder Übertragung.

Nutzung, Lizenz und KI

Brauche ich ein Nutzerkonto?

Ja. Für Trial, Kauf, Lizenzprüfung und BuiltSmart AI Credits ist ein Nutzerkonto erforderlich. Der Login erfolgt per Magic Link.

Muss ich etwas installieren?

Nein. Die App läuft browserbasiert in modernen Browsern auf Windows, macOS und anderen Systemen.

Was ist in der Jahreslizenz enthalten?

Die Jahreslizenz enthält den App-Zugang für einen persönlichen Nutzerzugriff. KI-Nutzung wird separat über BuiltSmart AI Credits oder über einen eigenen Anthropic API Key abgerechnet.

Ist die KI-Nutzung in der Jahreslizenz enthalten?

Die Jahreslizenz enthält den App-Zugang. KI-Aktionen werden separat über BuiltSmart AI Credits oder über einen eigenen Anthropic API Key genutzt.

Wie funktionieren BuiltSmart AI Credits?

BuiltSmart AI Credits gehören zum jeweiligen Nutzerkonto und sind appübergreifend innerhalb dieses Nutzerkontos nutzbar. Vor KI-Aktionen zeigt die App den voraussichtlichen Credit-Verbrauch an.

Kann ich einen eigenen API Key nutzen?

Ja. Alternativ zu BuiltSmart AI Credits kann ein eigener Anthropic API Key genutzt werden. Die Abrechnung erfolgt dann direkt über Anthropic. Der Key bleibt lokal in der jeweiligen App und wird nicht an das BuiltSmart Lizenzsystem gesendet.

Werden meine Dokumente für KI-Training verwendet?

Bei Nutzung über die Anthropic API werden deine Inhalte nur zur jeweiligen KI-Anfrage übertragen. Nach Anthropic-Angaben werden Daten aus kommerziellen Produkten wie der Anthropic API nicht zum Training generativer Modelle verwendet, sofern der Kunde nicht ausdrücklich einer Nutzung zur Modellverbesserung zustimmt.

Kann ich mehrere Lizenzen kaufen?

Ja. Mehrere Lizenzen derselben App können im Checkout über die Menge gekauft werden. Jede Lizenz ist für einen getrennten persönlichen Nutzerzugriff vorgesehen.

Wie werden KI-Guthaben bei mehreren Lizenzen berechnet?

Mehrere App-Lizenzen schalten mehrere persönliche Nutzerzugriffe frei. BuiltSmart AI Credits werden separat pro Nutzerkonto geführt. Trial-Credits werden beim Start des Trials dem jeweiligen Nutzerkonto gutgeschrieben. Bezahlte AI-Credits stehen dem Nutzerkonto zur Verfügung, über das sie gekauft wurden.

Gibt es geteilte Team- oder Cloud-Arbeitsbereiche?

Nein. Mehrere Lizenzen bedeuten mehrere getrennte persönliche Nutzerzugriffe. App-Inhalte werden nicht automatisch zwischen Nutzern synchronisiert.

Kann ich mehrere unterschiedliche Apps gemeinsam kaufen?

Nein. Ein Checkout gilt jeweils für eine App. Mehrere Lizenzen derselben App können über die Menge gekauft werden.

Wo bleiben App-Inhalte?

App-Inhalte bleiben in der jeweiligen App lokal im Browser, soweit die App so gebaut ist. Wenn die App eigene Inhalte speichert, wird Export und Import angeboten.

Wie funktioniert der Trial?

Der Trial läuft 3 Tage und bietet vollen App-Zugriff. 1.000 app-spezifische BuiltSmart AI Test-Credits werden beim Start des Trials gutgeschrieben.
